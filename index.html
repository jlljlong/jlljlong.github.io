protocal_cegma

发表于 2016-05-17 | 分类于 Protocal

Overview

CEGMA (Core Eukaryotic Genes Mapping Approach) 是 Korf lab 发布的众多优秀程序中的一个，在该实验室2007年发表的文章中，作者从六种模式生物(Homo sapiens, Drosophila melanogaster, Arabidopsis thaliana, Caenorhabditis elegans, Saccharomyces cerevisiae and Schizosaccharomyces pombe)的蛋白序列中识别出了一个包含458个保守蛋白的数据集并且发布了CEGMA mapping procedure，结合这两者，可以被用来准确识别新测序基因组的exon-intron结构，这个方法不是用来注释整个基因组的，而是提供一个可靠的数据集给其他预测软件作为他们的训练集[1]。

而在2009年，该实验室的作者又发表了另外一篇文献，在原来458个保守蛋白的基础上，用复杂的方法提取了248个核心基因，用来推断序列组装的完整性。

官方网站：http://korflab.ucdavis.edu/datasets/cegma/

CEGMA procedure 的安装

Requirement

  • PERL
  • WU-BLAST
  • HMMER
  • GeneWise
  • geneid

Reference

  1. Genis Parra, Keith Bradnam and Ian Korf. CEGMA: a pipeline to accurately annotate core genes in eukaryotic genomes.”
    Bioinformatics, 23: 1061-1067 (2007)
  2. Genis Parra, Keith Bradnam, Zemin Ning, Thomas Keane, and Ian Korf. Assessing the gene space in draft genomes”
    Nucleic Acids Research, 37(1): 298-297 (2009)

Instruction

之前用新浪博客写过一段时间的blog，可是上面太多的广告真是让人厌烦。而且拍版也很成问题，写出来的内容一点也不好看。后来看了很多人用github搭建自己的博客，觉得页面简介、排版工整，很是喜欢。最后就决定用hexo＋github这个组合来搭建我的博客了。

不过在这里我还是要强烈推荐简书，这款博客应用。如果你不想自己动手搭建博客引擎的话，这个绝对是独一无二的代替。另外最新发现了一款云笔记leanote，既可写笔记，又可做博客。

Install

安装方法网上一大堆，我在这里不再浪费版面。下面列举了我安装中出现的问题供大家参考：（此部分会保持更新）

  • 在创建Github仓库时，仓库名一定要与用户名相同。
  • Github仓库不能有任何文件，包括readme文件，否则无法同步本地的hexo blog到Github。
  • 使用hexo d同步本地blog到github服务器，一定会显示INFO Deploy done: git，如果不报错，也什么都不显示的情况，那很有可能是没有安装hexo-git插件，用以下的命令安装npm install hexo-deployer-git --save。
